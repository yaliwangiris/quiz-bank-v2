# 律師一試考題專家

台灣律師第一試歷屆考題線上練習平台。純靜態網頁，無需安裝任何套件，開啟即用。

**線上使用：** [GitHub Pages 連結](https://edward-playground.github.io/quiz-bank-v2/)

## 功能

- **依科目練習** — 跨年度混合隨機出題，專攻特定科目
- **依年份練習** — 選擇特定年度與科目，完整作答該份考卷
- **即時回饋** — 答對彩紙動畫、答錯哭哭貓動畫
- **單選 / 複選題** — 自動識別題型，複選題標示提醒
- **筆記功能** — 每題可寫筆記，儲存於瀏覽器 localStorage
- **結果統計** — 答對數、正確率、各科正確率分佈
- **響應式設計** — 手機、平板、桌機皆可使用

## 題庫範圍

| 科目 | 代碼 | 涵蓋內容 |
|------|------|----------|
| 刑法 | 1301 | 刑法、刑事訴訟法、法律倫理 |
| 憲法 | 2301 | 憲法、行政法、國際公法、國際私法 |
| 民法 | 3301 | 民法、民事訴訟法 |
| 商法 | 4301 | 公司法、保險法、票據法、證券交易法、強制執行法、法學英文 |

年度：民國 104 年 ~ 111 年，共 32 份題庫、約 1500 題。

## 技術架構

```
index.html    ← 頁面結構
style.css     ← 樣式
app.js        ← 邏輯
bank/         ← 題庫 JSON 檔案 + manifest.json
```

純 HTML + CSS + JS，無框架、無建置步驟、無 npm 依賴。直接部署至 GitHub Pages 即可運作。

## 本地開發

用任何靜態伺服器開啟即可：

```bash
npx serve .
```

然後開啟 `http://localhost:3000`。

## 部署

GitHub repo → **Settings → Pages → Source** → 選擇 `main` branch、`/ (root)` → Save。
